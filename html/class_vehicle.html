Detailed Description

Abstract base class representing a generic vehicle.

The Vehicle class serves as a base class for specific vehicle types such as cars, buses, motorcycles, and trucks. It includes common attributes like VIN, model, kilometers driven, and brand. The class enforces implementation of specific methods in derived classes through pure virtual functions.
