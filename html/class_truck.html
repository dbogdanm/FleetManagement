Detailed Description

Represents a truck, derived from the Vehicle class.

The Truck class extends the Vehicle class by adding specific attributes such as load capacity and fuel efficiency. It overrides base class methods to provide truck-specific functionality and supports cloning, displaying, and maintenance operations.
