Detailed Description

Manages the fleet system including drivers, vehicles, and routes.

The FleetManager class provides functionality for managing a fleet, which includes adding, removing, and displaying drivers, vehicles, and routes. It supports dynamic memory management for vehicles and ensures deep copies are made for safe object handling.
